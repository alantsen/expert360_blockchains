##1. Think Thin (not Fat)

Internet

##Internet Apps (on the value extracted) - Generally, they can be categorised as 'fat' at the application layer and 'thin' at the protocol layer - Mainly due to the fact that applications have been able to re-aggregate data and create the value extracting network effects - This means that almost **ALL** of the value has accrued to the companies playing at the application layer in the stack. - Think Google with search and Facebook with social

Bitcoin (Permissionlees)

##Blockchain Apps (permissionless) - The outcome is reversed - Most of the value is moved to the shared protocol layer - This is mainly because: * there is a shared data layer (e.g. a blockchain); and * cryptographic tokens which have market value (e.g. bitcoin and eth). - This maybe why Bitcoin (the protocol, through the token) is worth more than all the Bitcoin startups - **COMBINED**.
